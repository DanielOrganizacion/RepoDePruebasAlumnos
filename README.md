REQUISITOS:
  -JDK Java 21
  -JUnit5

Para el que no se haya enterado aun de como generar los test de java aqui
los pasos a seguir para generarlos:

    - (Por si no estuviera hecho, que deberia estarlo)
        -Primero debeis crear un directorio en la raiz del proyecto en donde
        se van a almacenar los test, siendo el nombre el que querais siempre y
        cuando marqueis el archivo como "Test source roots"
    - (Para generar los test)
        -Debeis iros a la clase donde se alojan los metodos a los cuales les
        quereis generar un test, seguido de eso, presionais: ALT+Insert
        lo que abrirá una ventana con diferentes opciones, entre ellos
        aparecerá "Test..." (EL nombre de la opcion puede variar dependiendo
        de la version o extensiones usados en intellij)

        -A continuacion se os abrirá una ventana donde deberia salir la direccion
        en donde se van a alojar los test, las casillas desmarcadas de los metodos
        a los que les quieres hacer un test y el JUnit (Importante):
        
        ¡¡¡PARA EVITAR PROBLEMAS DE COMPATIBILIDAD SE USARÁ LA VERSION MAS RECIENTE
        DE JUNIT (JUNIT 5) PORFAVOR ELEGID LA VERSION JUNIT 5, Y EN EL CASO DE QUE
        OS APAREZCA EL BOTÓN "FIX O ARREGLAR" PULSADLO, SINÓ, NO SE OS APLICARÁ EL 
        FRAMEWORK AL IDE Y TENDREIS ERRORES CON TODOS LOS TESTS!!!
        
        -Una vez pulsado, os vais a la parte de los metodos y marcad la casilla de 
        VUESTRO test y generadlo.
    
    -A partir de este punto se os generará una clase test en la carpeta donde se envian
    los test para que los podais probar con el uso de las librerías ya aprendidas con Manolo.
    
    Solo os pido que seais ordenados y que tengais cuidado con la redundancia para evitar 
    conflictos a la hora de fusionar las ramas.
